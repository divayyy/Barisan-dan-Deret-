Pendahuluan

Barisan adalah urutan bilangan yang tersusun menurut pola tertentu. Jika setiap suku dijumlahkan secara berurutan, hasilnya disebut deret. Barisan dan deret terdiri atas dua jenis utama: Aritmetika dan Geometri. Materi ini akan membahas konsep, rumus, dan contoh perhitungan lengkap.

1. Barisan Aritmetika (BA)

Barisan aritmetika adalah barisan bilangan di mana setiap suku setelah suku pertama diperoleh dengan menambahkan bilangan tetap b pada suku sebelumnya.

Rumus Suku ke-n

Un = a + (n - 1) × b

Contoh Perhitungan

Suku pertama a = 3, beda b = 2. Hitung suku ke-5.

U5 = 3 + (5 - 1) × 2 = 3 + 8 = 11

2. Barisan Geometri (BG)

Barisan geometri adalah barisan bilangan di mana setiap suku setelah suku pertama diperoleh dengan mengalikan suku sebelumnya dengan bilangan tetap r.

Rumus Suku ke-n

Un = a × rn-1

Contoh Perhitungan

Suku pertama a = 2, rasio r = 3. Hitung suku ke-4.

U4 = 2 × 3³ = 2 × 27 = 54

3. Deret Aritmetika (DA)

Deret aritmetika adalah jumlah dari n suku pertama suatu barisan aritmetika.

Rumus Jumlah n Suku Pertama

Sn = n/2 × (a + Un)

Atau:

Sn = n/2 × [2a + (n-1) × b]

Contoh Perhitungan

Suku pertama a = 3, beda b = 2, jumlah 5 suku pertama.

S5 = 5/2 × (3 + 11) = 35

4. Deret Geometri (DG)

Deret geometri adalah jumlah dari n suku pertama suatu barisan geometri.

Rumus Jumlah n Suku Pertama (r ≠ 1)

Sn = a × (rn - 1) / (r - 1)

Contoh Perhitungan

Suku pertama a = 2, rasio r = 3, jumlah 4 suku pertama.

S4 = 2 × (3⁴ - 1) / (3 - 1) = 80

Penutup

Materi barisan dan deret ini memberikan dasar untuk memahami pola bilangan, perhitungan jumlah suku, dan penerapan matematika dalam berbagai konteks.
